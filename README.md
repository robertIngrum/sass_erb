# SassErb

Makes Sass and ERB get along by passing erb code directly to the CSS so that it can be compiled at runtime.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sass_erb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sass_erb
